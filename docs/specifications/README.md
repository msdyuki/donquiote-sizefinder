# プロジェクト仕様書

ここは「ドン・キホーテ（PPIH様）デジタル採寸サイネージ」プロジェクトの仕様を格納・管理するためのディレクトリです。

## 今後格納されるドキュメントの例

- **要件定義書**: システムの目的や必須機能、非機能要件など
- **画面設計書 (UI/UX)**: 画面遷移図や各画面のワイヤーフレーム
- **システムアーキテクチャ設計**: 使用する技術スタック、フロントエンド/バックエンド構成
- **API仕様書**: 外部システム（採寸データAPIなど）との連携仕様
- **デザインガイドライン**: テーマカラー、フォント、アニメーションのルール

※ 新しい仕様書を作成する際は、このディレクトリ (`docs/specifications/`) 内に Markdown ファイル等で追加してください。
